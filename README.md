🔮 Unicode Character Value Finder

Un buscador simple, moderno y minimalista de puntos de código (Unicode) para caracteres y emojis. Diseñado con una interfaz oscura basada en la paleta de colores Catppuccin Mocha.

🚀 Características

Soporte Completo para Emojis: Utiliza codePointAt(0) en lugar de charCodeAt para asegurar el procesamiento correcto de emojis y caracteres que requieren pares de sustitución (surrogate pairs).

Diseño Catppuccin Mocha: Interfaz oscura y elegante basada en la famosa paleta de colores para programadores.

Responsive: Totalmente adaptado para pantallas móviles, tablets y ordenadores.

Micro-interacciones: Animaciones fluidas en botones e inputs para mejorar la experiencia de usuario.
